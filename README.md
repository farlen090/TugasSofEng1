# Toy - Aplikasi Manajemen Tugas Kuliah

Aplikasi **Toy** adalah sistem manajemen tugas kuliah sederhana yang dibangun menggunakan Node.js dengan menerapkan metode **Test-Driven Development (TDD)** untuk memastikan kualitas kode dan fungsionalitas fitur berjalan dengan baik. Fitur utama aplikasi ini berfokus pada validasi form input tugas dan penentuan tingkat prioritas otomatis berdasarkan sisa hari deadline.
